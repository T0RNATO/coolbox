Schoolbox Fix Extension

A small Chrome/Edge extension to reskin and change some things about the Schoolbox homepage. Includes:
  • Coloured Tiles
  • Live Updating Subjects
  • Timer until period is over
  • Reordering of some elements
  • More soon.. hopefully

Installation Guide:

  1. Go to the GitHub repository.
  2. Click on the file extension.zip. This will download a file to your computer.
  3. Save it somewhere safe (Like in OneDrive).
  4. Unzip the folder.
  5. Go to chrome://extensions by pasting it into your address bar.
  6. In the top right (or middle left for Edge), enable the Developer Mode toggle.
  7. Click Load Unpacked and select your newly unzipped folder, which should have a couple files in it.
  8. Success!
